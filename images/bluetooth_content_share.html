To add sound effects in JavaScript after clicking a button, you can make use of the HTML5 Audio API. Here's what you can do:

First, create an audio element in your HTML with an id that you can use to reference it in JavaScript. For example:
<audio id="click-sound" src="click.mp3"></audio>
In this example, we're using an audio element with id="click-sound" and src="click.mp3". You can replace click.mp3 with the filename of your audio file.

Next, add an event listener to the button element in your JavaScript code to play the audio when the button is clicked. For example:
const button = document.querySelector('button');
const clickSound = document.querySelector('#click-sound');

button.addEventListener('click', () => {
  clickSound.currentTime = 0;
  clickSound.play();
});
In this example, we're using the querySelector method to select the button element and the audio element with id="click-sound". We're then adding an event listener to the button element that plays the audio file when the button is clicked.

The currentTime property is set to 0 to ensure that the audio file starts from the beginning each time the button is clicked.

That's it! Now your button should play a sound effect when clicked.
